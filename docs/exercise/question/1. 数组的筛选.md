# 1. 数组的筛选

> 从一个给定数组中筛选出值为偶数的项

思路：使用ES6中的新方法filter(), return出每一个除以2余数为0的项
```js
var arr = [1, 2, ,3 ,4, 5, 6];
var test = [];
test = arr.filter(ele=>{
    if(ele % 2 == 0) {
        return ele
    }
})
console.log(test);
```

>判断是否为质数
```js
function isPrime(num) {
    var res = [2, 3, 5, 7].some(ele => {
        if( num % ele == 0 ){
            return true;
        }
    })
    res == true? console.log('非质数'): console.log('质数');;
}
isPrime(21);
```
